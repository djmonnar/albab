👥
알밥에는 3가지 역할이 있어요
역할에 따라 볼 수 있는 화면이 달라요
🏪
사장님
  • 직원 등록 및 관리
  • 월별 급여 정산 확인
  • 1년치 급여 내역 조회
  • 직원 계좌 확인
  • 엑셀 파일 다운로드
  • 근무 기록 수정/삭제
👤
직원
  • 출퇴근 시간 직접 입력
  • 내 근무 기록 확인
  • 예상 급여 실시간 계산
  • 기록 수정 가능
  • 내 계좌 정보 입력
  • 월별 내 기록 조회
💡
앱 설치가 필요 없어요!
주소를 크롬에서 열면 바로 사용할 수 있어요. 스마트폰 홈 화면에 추가하면 앱처럼 쓸 수 있어요.
🏪
사장님 시작하기
처음 한 번만 설정하면 돼요!
1
알밥 접속 → "가게" 탭 → "새 가게 등록"
처음 쓰시는 분은 새 가게를 등록해야 해요.
로그인 화면에서 "✨ 새 가게 등록" 버튼을 눌러주세요.
2
가게 정보 입력
가게 이름 — 실제 가게 이름 (예: 담담)
가게 코드 — 영문+숫자 조합, 직원들이 로그인할 때 쓸 코드 (예: DAMDAM)
비밀번호 — 사장님 전용 비밀번호
💡 가게 코드는 영문 대문자로 짧게 만드세요 (예: DAMDAM, CAFE01)
3
직원 등록하기
로그인 후 "직원" 탭에서 직원을 추가해요.
직원 이름, 시급, PIN(4자리 숫자)을 입력하면 끝!
🔢 PIN은 직원이 로그인할 때 쓰는 비밀번호예요
4
직원에게 알려줄 정보 전달
직원에게 이 3가지를 알려주세요:
① 알밥 주소 — djmonnar.github.io/albab
② 가게 코드 — 내가 설정한 코드 (예: DAMDAM)
③ PIN 번호 — 직원별로 설정한 4자리 숫자
👤
직원 사용 방법
사장님께 코드 받은 후 이렇게 하세요
1
알밥 접속 → "직원" 탭 선택
로그인 화면에서 "👤 직원" 탭을 눌러요.
2
로그인 정보 입력
가게 코드 — 사장님께 받은 코드 입력
이름 — 사장님이 등록해 준 내 이름 입력
PIN — 사장님께 받은 4자리 숫자 입력
3
근무 기록 추가
홈 화면에서 날짜, 출근 시간, 퇴근 시간을 입력하고
"+ 기록 추가" 버튼을 눌러요.
입력하면 자동으로 근무시간과 예상 급여가 계산돼요!
⏰ 퇴근 후에 입력해도 되고, 그날그날 입력해도 돼요
4
내 정보 탭에서 계좌 입력
하단 "👤 내 정보" 탭에서 휴대폰 번호, 은행, 계좌번호를 입력하면
사장님이 급여 이체할 때 확인할 수 있어요.
✨
주요 기능 한눈에 보기
이런 것들이 가능해요
💰
급여 자동 계산
출퇴근 시간을 입력하면 시급 × 근무시간이 자동으로 계산돼요. 직원도 본인 예상 급여를 바로 확인할 수 있어요.
📅
1년치 기록 보관
월별 급여 합계가 자동으로 정리돼요. 작년 몇 월에 얼마 나갔는지도 쉽게 확인 가능해요.
📊
엑셀 다운로드
정산 페이지에서 버튼 한 번이면 직원별 급여 내역이 CSV 파일로 저장돼요. 엑셀에서 바로 열 수 있어요.
✏️
기록 수정 가능
잘못 입력한 시간은 언제든 수정하거나 삭제할 수 있어요. 직원 본인도, 사장님도 수정 가능해요.
💳
계좌 정보 관리
직원이 앱에서 자기 계좌번호와 은행을 입력하면 사장님이 급여 이체할 때 바로 확인할 수 있어요.
🔒
분리된 접근 권한
직원은 본인 기록만 볼 수 있어요. 다른 직원 급여나 정보는 절대 볼 수 없어요.
📊
엑셀로 정산하는 방법
월말 정산이 훨씬 간편해져요
1
사장님으로 로그인 → 정산 탭
하단 📊 정산 탭을 눌러요. 좌우 화살표로 원하는 달을 선택해요.
2
"📊 엑셀 다운로드" 버튼 클릭
버튼을 누르면 자동으로 파일이 다운로드돼요.
파일명 예시: 담담_2025-03_급여정산.csv
3
엑셀에서 열기
다운로드된 파일을 엑셀 또는 구글 스프레드시트에서 열면 직원별 상세 내역과 합계가 정리되어 있어요.
📅
연간 엑셀도 받을 수 있어요!
하단 📅 연간 탭에서 "📊 연간 엑셀" 버튼을 누르면 1월~12월 월별 급여 합계를 한 번에 받을 수 있어요.
❓
자주 묻는 질문
궁금한 게 있으면 먼저 확인해보세요
앱을 설치해야 하나요? ▼
설치 필요 없어요! 크롬 브라우저에서 주소를 입력하면 바로 사용할 수 있어요. 스마트폰 홈 화면에 추가하면 앱처럼 바탕화면 아이콘으로 쓸 수 있어요.

홈 화면에 추가하는 방법 (안드로이드): 크롬 → 우상단 점 3개 메뉴 → "홈 화면에 추가"
직원이 PIN을 잊어버렸어요 ▼
사장님 로그인 → 👥 직원 탭 → 해당 직원 "수정" 버튼 → 새 PIN 입력 후 저장하면 돼요. PIN 칸을 비워두면 기존 PIN이 유지돼요.
근무 시간을 잘못 입력했어요 ▼
직원: 📋 내 기록 탭 → 해당 날짜의 기록에서 "수정" 버튼을 눌러서 고칠 수 있어요.

사장님: 📊 정산 탭 → 직원 카드를 눌러 펼치면 기록별로 "수정"/"삭제" 버튼이 있어요.
여러 가게를 운영하는데 각각 따로 관리되나요? ▼
네! 가게마다 코드가 달라서 완전히 분리되어요. 가게 A 직원은 가게 A 코드로만 로그인 가능하고, 가게 B 데이터를 볼 수 없어요.
데이터가 사라지지 않나요? ▼
걱정 마세요! 모든 데이터는 구글 Firebase(클라우드)에 저장돼요. 폰을 바꾸거나 앱을 지워도 데이터는 그대로 유지돼요. 어디서든 로그인하면 기록이 나와요.
직원이 다른 직원 급여를 볼 수 있나요? ▼
절대 볼 수 없어요. 직원은 본인 기록과 예상 급여만 볼 수 있어요. 다른 직원의 시급이나 근무 기록은 철저히 분리되어 있어요.
지난달 기록도 확인할 수 있나요? ▼
네! 화면 상단의 ‹ › 화살표를 눌러서 원하는 달로 이동할 수 있어요. 1년치 이상 데이터가 보관돼요.
